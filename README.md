# Featured Projects

### Real-Time Environmental Streaming Pipeline
A real-time data engineering pipeline that simulates environmental sensor data streams and processes them using distributed streaming technologies. The system integrates Kafka ingestion, Spark Structured Streaming processing, Airflow workflow orchestration, and PostgreSQL storage.

**Technologies:**  
Python, Kafka, Spark, Airflow, PostgreSQL, Data Engineering

Repository:  
https://github.com/Cornelius-Mulyo/real-time-env-streaming-pipeline

---

### MRI Synthetic ML Pipeline
A synthetic MRI spectroscopy machine learning pipeline using FFT-based preprocessing, spectral shifting, and feature extraction to analyze frequency-domain signals from imaging data.

**Technologies:**  
Python, NumPy, SciPy, Signal Processing, Machine Learning

Repository:  
https://github.com/Cornelius-Mulyo/mri-synthetic-ml-pipeline

---

### YouthLink Opportunity Platform
A full-stack web platform that enables users to discover youth opportunities such as internships, scholarships, and workshops. The platform supports document upload, search filtering, and database-backed management.

**Technologies:**  
Python, Flask, SQLite, Bootstrap, Backend Development

Repository:  
https://github.com/Cornelius-Mulyo/youthlink-webplatform-v2

### Cassandra LangFlow Chatbot
A LangFlow-based chatbot system integrated with Apache Cassandra to manage conversational data and demonstrate NoSQL database design and query workflows.

**Technologies:**  
Python, Apache Cassandra, LangFlow, NoSQL

Repository:  
https://github.com/Cornelius-Mulyo/cassandra-langflow-chatbot
